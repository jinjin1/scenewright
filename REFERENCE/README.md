# REFERENCE/ — 도메인 지식 카드

이 폴더는 **드롭인 지식 카드** 모음입니다. `/synopsis`·`/script`가 대본에 어떤 개념·프레임워크가 등장하면 이 카드를 참조해 **정의·한계·예시의 정확성**을 확보합니다.

## 쓰는 법

- 당신 채널 주제에서 자주 나오는 개념마다 카드 한 장(`<개념>.md`)을 만든다.
- 카드엔 한 줄 정의 + 핵심 요소 + 흔한 오해/한계 + 짧은 예시를 담는다.
- 카드가 있으면 LLM이 그 개념을 헛디디지 않는다. 없으면 일반 지식으로 처리되니, **틀리면 곤란한 핵심 개념**부터 카드화한다.

## 예시 카드 (PM 채널)

리포에 포함된 `RICE.md`·`JTBD.md`는 **예시**입니다 — PM(프로덕트 매니지먼트) 주제의 카드. 다른 주제로 쓸 거면 교체하세요. 예:

- **과학 채널**: `RayleighScattering.md`, `Wavelength.md`
- **금융 리터러시**: `CompoundInterest.md`, `Diversification.md`
- **역사 채널**: 시대·사건별 팩트 카드
